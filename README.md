# Netacad-Ethical-Hacker-Capstone-Project
This repository contains the documentation, network maps, and custom scripts developed for the Cisco NetAcad Ethical Hacker Capstone. The project focuses on applying the Cyber Kill Chain and MITRE ATT&amp;CK frameworks to secure enterprise-grade network architectures. Focus Areas: Web Application Security, Password Cracking, and Network Sniffing.
